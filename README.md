# Retail Sales Analysis using Excel

## Overview
This project focuses on analyzing retail sales data using Excel to uncover business trends, profitability patterns, and actionable insights.

This project was completed under the guidance of Mr. Kushal Sharma (EY Trainer & Guest Lecturer).

## Objectives
- Analyze sales performance across regions, categories, and customer segments  
- Identify profit trends and loss-making areas  
- Understand the impact of discounts on profitability  
- Evaluate product and regional performance  

## Key Insights
- Certain regions contributed significantly more to overall sales  
- Some product categories generated high sales but low profit margins  
- Discounts had a noticeable negative impact on profitability  
- A small group of p- roducts contributed disproportionately to total revenue  

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Analysis)

## Project Structure
- data/: Excel dataset used for analysis  
- presentation/: PowerPoint presentation containing insights and visuals  

## Learnings
- Excel can be effectively used for business analysis and decision-making  
- Data visualization simplifies complex patterns  
- Insight-driven analysis is more valuable than raw data  

## Conclusion
This project demonstrates how structured data analysis using Excel can uncover meaningful insights and support better business decisions.
